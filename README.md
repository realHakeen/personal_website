## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- **Social Links**: Edit `src/components/cards/ProfileCard.tsx`
- **Experiences**: Edit `src/components/sections/Roadmap.tsx`
- **Projects**: Edit `src/components/sections/Projects.tsx`
- **Articles**: Edit `src/components/sections/Writings.tsx`
- **Books**: Edit `src/components/sections/Bookshelf.tsx`
